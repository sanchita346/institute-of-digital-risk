# Institute of Digital Risk — Brand & Homepage

> A brand identity and responsive single-page website built for the **Institute of Digital Risk (IDR)** — an industry-led training and deployment institute focused on digital, cyber, and AI risk.

**Live Demo →** https://github.com/sanchita346/institute-of-digital-risk.git

---

## Project Overview

This project was built as a design and development assignment to create a complete brand and homepage for IDR. The deliverables include a logo system (two variants) and a fully responsive, animated single-page website — built entirely with vanilla HTML, CSS, and JavaScript.

## Tech Stack

- **HTML5** — Semantic markup (`header`, `nav`, `main`, `section`, `article`, `footer`)
- **CSS3** — Custom properties, Flexbox, Grid, keyframe animations
- **Vanilla JavaScript** — No frameworks or libraries
- **Google Fonts** — Space Grotesk, Bebas Neue, JetBrains Mono
- **SVG** — Inline vector logo and cube illustration

> No Bootstrap. No Tailwind. No frameworks. Pure vanilla code.

## Features

- **Custom animated cursor** — orange dot with lagging ring that reacts to interactive elements
- **Particle network background** — 80 floating particles with orange connection lines
- **Animated 3D cube** — SVG isometric cube with floating orbit rings and data cards
- **Scroll reveal animations** — Elements fade and slide in as you scroll
- **Sticky navigation** — Glass blur effect triggered on scroll
- **Responsive mobile layout** — Hamburger menu, stacked grid on small screens
- **Marquee ticker** — Scrolling keyword strip that pauses on hover
- **Hover microinteractions** — Cards lift, buttons shimmer, nav links draw underline
- **Form validation** — Basic email check with animated success message
- **Section watermarks** — Large ghost numbers (01–04) in each section background

## Logo Design — Rationale

The cube icon was chosen to represent the layered, structured nature of digital risk — multiple faces suggesting different threat dimensions held together as one resilient form. A shield motif is embedded on the top face to directly reference risk governance and protection. Orange was selected as the primary accent to convey urgency and energy, balanced by black and grey for authority and credibility. The condensed uppercase wordmark ensures legibility at small sizes, from favicons to mobile headers.
